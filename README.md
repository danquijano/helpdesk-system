# HelpDesk System

A comprehensive full-stack ticketing system designed for IT support management. This application streamlines the process of reporting, tracking, and resolving technical issues within an organization.

## ✨ Features

- **User Authentication**: Secure login with role-based access (Employee, Technician, Admin)
- **Ticket Management**: Create, update, and track support tickets
- **Real-time Updates**: Status changes and comments update in real-time
- **Dashboard**: Personalized views for different user roles
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🛠️ Tech Stack

**Frontend:**
- React with TypeScript
- TailwindCSS for styling
- Axios for API calls
- React Router for navigation

**Backend:**
- Node.js with Express
- RESTful API architecture
- JWT authentication
- MongoDB for database (coming soon)

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Git

## 🚀 Getting Started

1. Clone the repository
2. Install backend dependencies: `cd backend && npm install`
3. Install frontend dependencies: `cd frontend && npm install`
4. Start the backend server: `cd backend && npm run dev`
5. Start the frontend app: `cd frontend && npm start`
6. Visit `http://localhost:3000` to see the application

## 👥 User Roles

- **Employee**: Create tickets and track their status
- **Technician**: View assigned tickets and update resolution progress
- **Admin**: Full access to manage users, tickets, and system settings
